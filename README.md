# Employee Performance Dashboard (Excel)

## 📌 Project Overview
This project analyzes employee attendance, salary, overtime, and department performance using Microsoft Excel.

## 📊 Features
- Department-wise Salary Analysis
- Attendance Analysis
- Employee Distribution
- Top 5 Employees by Overtime Hours
- KPI Dashboard

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- IF Formula
- INDEX + MATCH
- Data Cleaning
- Dashboard Design

Built an interactive Excel dashboard to monitor employee performance and provide actionable business insights.
